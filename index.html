Nossa trajetória

Durante o curso do CPDI - Além de jogar eu faço jogos, estudamos os conceitos basicos de programação, onde usamos a linguagem Lua para criação de jogos e Solar 2D para rodar nossos scripts.
Após concluirmos o jogo estudamos sobre a linguagem WEB voltada para parte visual (Front End) usando HTML5 e CSS3. Estuamos os conceitos principais da linguagem para que assim tenhamos como criar nossa página web com as informações de para disponibilizamos para download o jogo.
